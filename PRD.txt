PRD — Site vitrine “Le Comptoir des Étuves”
1. Résumé du projet
Créer un site web vitrine pour Le Comptoir des Étuves, avec 2 pages principales :
1.	une page de présentation du snack,
2.	une page pour consulter la carte de façon claire et dynamique.
Le site doit donner envie de venir sur place, aider le référencement local sur Google, et permettre aux clients d’accéder rapidement à la carte, y compris depuis un QR code posé sur les tables.
Le site doit être conçu d’abord pour mobile.
Le téléphone est le support principal.
La version desktop est une adaptation de la version mobile, et non l’inverse.
Nom du projet : Le Comptoir des Étuves
________________________________________
2. But du site
Le site doit servir à :
•	présenter clairement le snack,
•	rassurer les visiteurs,
•	montrer les informations utiles dès les premières secondes,
•	afficher la carte de façon agréable et rapide,
•	aider les clients à commander ou venir sur place,
•	améliorer la visibilité sur Google,
•	mettre en avant la disponibilité sur Uber Eats.
Le site doit aussi favoriser les actions rapides sur mobile :
•	voir la carte,
•	appeler,
•	ouvrir l’itinéraire,
•	commander,
•	changer la langue si besoin (automatique via google si possible ?)

Le site doit être pensé d’abord pour smartphone.
Règles de base
•	lecture verticale simple,
•	peu de texte visible d’un coup,
•	gros boutons faciles à toucher,
•	navigation courte,
•	temps de chargement rapide,
•	informations utiles visibles très vite,
